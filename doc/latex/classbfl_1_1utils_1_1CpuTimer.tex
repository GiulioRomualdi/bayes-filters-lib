\doxysubsection{Detailed Description}
\subsubsection*{template$<$typename timetype = std\+::chrono\+::milliseconds$>$\newline
class bfl\+::utils\+::\+Cpu\+Timer$<$ timetype $>$}

This template class provides methods to keep track of time. 

The default time unit is milliseconds, but can be changed during object creation using a different std\+::chrono\+::duration type. See \href{https://en.cppreference.com/w/cpp/chrono/duration}{\texttt{ https\+://en.\+cppreference.\+com/w/cpp/chrono/duration}} for reference. The timer starts and stops using, respectively, \mbox{\hyperlink{classbfl_1_1utils_1_1CpuTimer_aafb5ec4ed5936f14bb066d9e809f1f0e}{start()}} and \mbox{\hyperlink{classbfl_1_1utils_1_1CpuTimer_ad8e710e2a21b97c5e13ff2f338445a96}{stop()}} method. The get the time elapsed between a \mbox{\hyperlink{classbfl_1_1utils_1_1CpuTimer_aafb5ec4ed5936f14bb066d9e809f1f0e}{start()}} and \mbox{\hyperlink{classbfl_1_1utils_1_1CpuTimer_ad8e710e2a21b97c5e13ff2f338445a96}{stop()}} call, use the \mbox{\hyperlink{classbfl_1_1utils_1_1CpuTimer_ac7702cd9061b12db77c9e249d7b77046}{elapsed()}} method. The \mbox{\hyperlink{classbfl_1_1utils_1_1CpuTimer_ac7702cd9061b12db77c9e249d7b77046}{elapsed()}} method can be used to get the elapsed time from the \mbox{\hyperlink{classbfl_1_1utils_1_1CpuTimer_aafb5ec4ed5936f14bb066d9e809f1f0e}{start()}} call, before calling \mbox{\hyperlink{classbfl_1_1utils_1_1CpuTimer_ad8e710e2a21b97c5e13ff2f338445a96}{stop()}}. To get the absolute time from epoch use the \mbox{\hyperlink{classbfl_1_1utils_1_1CpuTimer_adf29a51b31dd9442679c1f97b7c31db4}{now()}} method.
