◆ update_weights_online_

const bool bfl::UKFCorrection::update_weights_online_ = false
protected

The unscented transform for a generic non-additive measurement model requires the computation of 2 * n + 1 sigma points where n depends not only on the size of the input state but also on the size of the noise.

While the state size is fixed, the noise size might depend on the number of measurements available, hence n might change online. Since the unscented transform weights, stored in ut_weight_, depend on n the user can specify, using the ctor input parameter update_weights_online, that the weights have to be re-computed at each iteration.
